# Code Of Conduct

## Our Commitment

We are committed to making participation in this project respectful, safe, and welcoming for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Expected Behavior

Examples of behavior that contribute to a positive environment include:

1. Being respectful and considerate in communication.
2. Giving and accepting constructive feedback.
3. Focusing on what is best for the community and project users.
4. Showing empathy toward other contributors.

Examples of unacceptable behavior include:

1. Harassment, intimidation, or discriminatory language.
2. Personal attacks, trolling, or insulting comments.
3. Publishing private information without explicit permission.
4. Any conduct that could reasonably be considered inappropriate in a professional setting.

## Scope

This Code of Conduct applies in all project spaces, including:

1. GitHub issues and pull requests.
2. Project discussions and documentation channels.
3. Public and private communications representing the project.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may take corrective action in response to behavior that is inappropriate, threatening, offensive, or harmful.

If you experience or witness unacceptable behavior, report it privately to project maintainers via the security reporting path in `SECURITY.md`.
