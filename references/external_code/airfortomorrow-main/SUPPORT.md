# Support

Use this guide to route usage questions, bug reports, and security concerns to
the right channel.

## Usage Questions and Troubleshooting

For setup help, pipeline usage questions, and expected-behavior clarifications:

- Open a GitHub issue: https://github.com/unicef/airfortomorrow/issues
- Include your run command, relevant logs, and environment details (OS, Docker
  version, and branch/commit) so maintainers can reproduce quickly.

## Bug Reports

For confirmed defects or regressions:

- Open a bug report issue using the repository bug template:
  https://github.com/unicef/airfortomorrow/issues/new/choose
- Provide minimal reproduction steps, expected vs. actual behavior, and any
  relevant data paths or stack traces.

## Security Vulnerabilities

Do **not** report security vulnerabilities in public GitHub issues, PRs, or
discussions.

Report suspected vulnerabilities privately following
[SECURITY.md](SECURITY.md). Public disclosure before a fix can put users and
operators at risk.
